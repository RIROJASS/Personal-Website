Auyan Tepui Analysis

Using Google Earth Engine and remote sensing technology, I analyzed the unique ecosystem of Auyan Tepui, one of Venezuela's iconic table mountains.

Project Highlights:

  • Implemented vegetation classification algorithms to identify endemic plant species zones
  • Analyzed seasonal rainfall patterns and their impact on the tepui's waterfalls, including Angel Falls
  • Created interactive visualizations of elevation profiles and ecological zones
  • Combined satellite imagery with traditional ecological knowledge

Technologies used: Google Earth Engine, Python, QGIS, Remote Sensing

SHPE National Convention Corporate Database

As the Corporate Database Director for the Society of Hispanic Professional Engineers (SHPE), I developed and maintained a comprehensive system for tracking corporate partnerships and engagement during the National Convention.

Project Highlights:

  • Designed and implemented a relational database to track over 200 corporate partners and their engagement with SHPE members
  • Created an analytics dashboard to visualize recruitment metrics and partnership outcomes
  • Developed automated reporting tools to provide real-time insights during the convention
  • Implemented data security protocols to protect sensitive partnership information

This project bridged my technical database skills with my leadership role in SHPE, creating valuable tools that enhanced the organization's ability to foster meaningful connections between Hispanic engineering students and industry partners.

Technologies used: SQL, Python, Tableau, Excel

AI-Enhanced Remote Sensing (NASA Innovation Challenge)

As part of the NASA x Blackstone Innovation Challenge, I explored applications of NASA's technology for real-world problem-solving, with a focus on archaeological discovery.

Project Highlights:

  • Researched applications of the Groff Algorithm for pattern recognition in satellite imagery
  • Developed a framework for identifying potential archaeological sites using machine learning and remote sensing
  • Investigated NASA patents related to sustainability and climate analysis
  • Explored triangulation capabilities for location-based pattern recognition

This project represents the intersection of cutting-edge technology and archaeological research, demonstrating how AI can enhance our ability to discover and protect cultural heritage sites through non-invasive means.

AI Archaeology Classification Tool

Developed a machine learning system to classify archaeological artifacts from photographic data, integrating computer vision with archaeological expertise.

Project Highlights:

  • Created a dataset of over 5,000 artifact images across multiple categories
  • Trained a convolutional neural network to identify pottery types, tool materials, and approximate age
  • Developed a user-friendly interface for field researchers to use during excavations
  • Achieved 87% accuracy on test datasets

Technologies used: Python, TensorFlow, Keras, OpenCV, Flask
